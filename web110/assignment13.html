Single-Table SELECT Statements

Below are some examples of single-table SELECT statements:

  • SELECT * FROM CUSTOMERS;
  • SELECT customer_name, city FROM CUSTOMERS WHERE country='USA';
  • SELECT * FROM ORDERS WHERE order_date BETWEEN '2022-01-01' AND '2022-12-31';
  • SELECT product_name, unit_price FROM PRODUCTS ORDER BY unit_price DESC;
  • SELECT COUNT(*) AS total_customers FROM CUSTOMERS;

INSERT, UPDATE, and DELETE Statements

Below are some examples of INSERT, UPDATE, and DELETE statements:

  • INSERT INTO CUSTOMERS (customer_name, city, country) VALUES ('John Doe', 'New York', 'USA');
  • UPDATE CUSTOMERS SET city='Los Angeles' WHERE customer_id=123;
  • DELETE FROM CUSTOMERS WHERE customer_id=456;
